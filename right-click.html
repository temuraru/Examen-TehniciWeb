Right-click context menu

There are 3 actions implemented for now:
  • View - which takes you to the target element's href URL
  • Edit - which edits the 'label' name for the target element
  • Delete - which deletes the target element
You can choose the contextual menu theme by using the selector in the top-right corner:
  • Bright
  • Light
  • Dark
  • Plain
  • I. Aplicatii (Jocuri): [n x n] Slider puzzle
  • II. Controale: Right-click - contextual menu
